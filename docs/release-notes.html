Release Notes

1.01.00

  • Externalizing JTidy properties file in order to allow users to set any JTidy configurations (e.g, adding unknown tags which cannot be tidied by default).
    Thanks to Eray Konur!
  • Fixing documentation about how to set font files in HtmlPDFRenderingFilter filter configuration in web.xml
  • Upgrade the demo project to base itself on Hippo CMS 7.9 (but the module should work on Hippo CMS 7.8, too.)

0.01.01

  • Fixing License Headers

0.01.00

  • Initial Release
  • HTML to XHTML transformation by using JTidy
  • Transforming image tags (<img>) to use local absolute URLs in order to embed images in the generated PDF output
  • Transforming link tags (<a>) to use external absolute URLs in order to enable linking in the generated PDF output
  • Transforming XHTML to PDF by using Flying Saucer
  • Site documentation
